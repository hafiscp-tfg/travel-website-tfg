# **App Name**: Sharafiya Travel Hub

## Core Features:

- Flight Search & Booking: Enable users to search for domestic and international flights using a real-time API (Amadeus/Sabre/Galileo or consolidator like TBO Holidays) and complete bookings with integrated payment gateway (Razorpay/Cashfree/PhonePe) and instant ticket generation.
- Visa Inquiry Form: A form for users to inquire about visa services, specifically for Gulf, Schengen, US/UK visas. The form will collect necessary information for Sharafiya to initiate the visa application process. Form sends the info using email.
- Tour Package Catalog: Display pre-designed tour packages (Kerala & International) with descriptions, images, and inclusions/exclusions, and a 'Request Quote' form.
- Content Generation tool for Tour Packages: Use generative AI to create compelling descriptions for tour packages based on destination, duration, and activities.  AI will use a tool to find attractions, amenities and details that match tour criteria, from external websites
- Admin Dashboard: A simple admin panel to view booking details and manage inquiry emails, manage tour packages, and update visa information (using Strapi headless CMS).
- WhatsApp Integration: Include a WhatsApp link for direct communication, catering to the preferences of the Kerala market.

## Style Guidelines:

- Primary color: Deep blue (#2962FF), evocative of a calm sea and Kerala's waters.
- Background color: Very light blue (#E7EBFD), offering a clean, uncluttered, airy feeling.
- Accent color: Purple (#A64CFF), used on buttons to create emphasis and contrast.
- Headline font: 'Playfair', serif, to be used for headlines, titles, and limited body copy.
- Body font: 'PT Sans', sans-serif, to be used for article content and body copy.
- Use clean, modern icons to represent various services and destinations.
- Employ a grid-based layout for consistent alignment and responsiveness.
